# Top 5 European Leagues Attacking Returns Dashboard

This project is an interactive Power BI dashboard analyzing attacking returns across the top five European football leagues for the 2024/2025 season.

The dashboard focuses on player attacking performance, including goals and league-based comparisons across:

- Premier League
- La Liga
- Serie A
- Bundesliga
- Ligue 1

## Project Overview

The goal of this project is to provide a clear visual analysis of attacking output across Europe’s top leagues. The dashboard allows users to compare players, clubs, and leagues based on goal-scoring performance and attacking contribution.

## Key Features

- Overview page comparing all top five leagues
- Separate pages for each league
- Top scorers analysis
- Goals by league and club
- Player-level attacking statistics
- Interactive filters and slicers
- FIFA-inspired dark visual theme

## Tools Used

- Power BI
- DAX
- Power Query
- Excel / CSV dataset
- GitHub

## Purpose

This dashboard was created as a final project to demonstrate data cleaning, data modeling, DAX calculations, and dashboard design using Power BI. It combines football analytics with interactive visual storytelling to make attacking performance easier to understand and compare.
